This method compares two strings lexicographically and returns an integer representing their relative order.

void main() {
 
  String str1 = "prabhu";

  String str2 = "prabhu";

  print("String 1 ${str1}");
  
  print("String 2 ${str2}");
  
  print(str1.compareTo(str2));
  
}

OUTPUT

String 1 prabhu
String 2 prabhu
0

OUTPUT

String 1 pra
String 2 prabhu
-1

OUTPUT

String 1 prabhu
String 2 pra
1
