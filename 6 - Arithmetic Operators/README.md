# 1 - Arithmetic Operators Table

No	Arithmetic Operators	Meaning
1	+	Addition
2	-	Subtraction
3	*	Multiplication
4	/	Division
5	%	Modulus

# 2 - Addition Example

Perform mathematical Addition

```
void main() {
    final int x = 2;
    
    final int y = 2;
    
    print("x = $x");
    
    print("y = $y");
    
    print("$x + $y = ${x+y}");
    
}
```

**Output**
```
x = 2
y = 2
2 + 2 = 4
```

# 3 - Subtraction Example

Perform mathematical Subtraction

```
void main() {
    final int x = 10;
    
    final int y = 2;
    
    print("x = $x");
    
    print("y = $y");
    
    print("$x - $y = ${x-y}");

```

**Output**
```
x = 10
y = 2
10 - 2 = 8
```

